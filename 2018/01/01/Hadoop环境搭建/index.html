Hadoop系列一：环境搭建

网络

  • 如何在shell中确认能否上网
    ping www.baidu.com
  • vm3种虚拟机上网方式的差异
    1. NAT：虚拟机中node机器连接在一起，与host机通过虚拟网关相连，并通过此网关上网。此网关会分配给host机一个新的网段。网上说，本机不能访问虚拟机，经测试，本机是可以访问虚拟机的。
    2. Bridge：虚拟机模拟与host并列的机器，与host机器所在网络使用一个网段
    3. HostOnly：node机器只与host机器相连，不能上网。
  • 配置
    1. 使用默认的NAT方式上网
    2. 在node机内，在/etc/sysconfig/network-script/ifcfg-ens33中，将最后的noboot改为yes，重启即可

主机名

  1. /etc/hosts
    增加主机的路由信息
  2. /etc/sysconf/network
    增加HOSTNAME=node名

界面

  1. 界面也是一种分层的结构，界面都都是调用的底层的X-Window协议，使用X-Lib API
  2. 中间层有Qt，再上一层是KDE。Qt收费。
  3. 中间层是GDK/GTK，再上一层是GNOME。Ubuntu默认使用。

JDK

[root@localhost ~]$ curl -O http://download.oracle.com/otn-pub/java/jdk/7u79-b15/jdk-7u79-linux-x64.rpm

2.使用rpm命令安装
[root@localhost ~]# rpm -ivh jdk-7u79-linux-x64.rpm

3.设置环境变量
[root@localhost java]# vi /etc/profile
在打开的profile文件中添加如下内容

  1. 让修改生效
    [root@localhost java]# source /etc/profile

  2. 验证
    java -version
